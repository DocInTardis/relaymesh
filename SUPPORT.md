# Support

## How to Get Help

1. Read `README.md` and `docs/` first.
2. Check existing issues before opening a new one.
3. For usage questions, open an issue with:
   - runtime version
   - OS and Java version
   - command used
   - relevant logs and outputs

## Security Issues

Use the process described in `SECURITY.md`.

Do not disclose sensitive details in public issues before triage.
